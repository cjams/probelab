# probelab

This is a library for linear probes. The library supports end-to-end
experiementation and finding probes over various features as desired
by the experimenter.

We should be able to reproduce different papers and their 
idosyncracies - like which tokens and layers they use to train
and intervene in.

Use best practices for finding probes and ensure that interfaces are 
ergnomic and friendly for experimentation
